🌱 2. 토양 시료 접수

토양 분석을 위한 시료를 접수합니다. 민원인 정보와 필지 정보를 입력하세요.

토양 시료 접수 화면

접수 방법

  1. 접수일자를 선택합니다 (기본값: 오늘)
  2. 민원인 정보(성명, 전화번호, 주소)를 입력합니다
  3. 구분(논, 밭, 과수, 시설)을 선택합니다
  4. 목적(일반재배, 무농약, 유기, GAP, 저탄소)을 선택합니다
  5. 필지 추가 버튼으로 필지 정보를 입력합니다
  6. 등록 버튼을 클릭하여 접수를 완료합니다

💡 팁

필지 주소를 입력할 때 "봉화"를 입력하면 봉화군의 리 목록이 자동완성됩니다.

목록 보기

상단의 "목록" 버튼을 클릭하면 접수된 시료 목록을 확인할 수 있습니다.

토양 시료 목록 화면

🧫 3. 잔류농약 시료 접수

농산물 잔류농약 검사를 위한 시료를 접수합니다.

잔류농약 시료 접수 화면

접수 방법

  1. 접수일자와 민원인 정보를 입력합니다
  2. 농산물명을 선택합니다 (작물 검색 가능)
  3. 재배지 주소를 입력합니다
  4. 출하 예정일을 선택합니다
  5. 인증 종류(무농약, 유기, GAP, 저탄소, 일반)를 선택합니다
  6. 등록 버튼을 클릭합니다

💡 팁

여러 건의 시료를 한 번에 접수하려면 "의뢰 추가" 버튼을 사용하세요.

💧 4. 수질 시료 접수

지하수, 농업용수 등 수질 검사를 위한 시료를 접수합니다.

수질 시료 접수 화면

접수 방법

  1. 시료명(지하수, 하천수, 저수지, 수돗물 등)을 선택합니다
  2. 채취장소(리+지번)를 입력합니다
  3. 검사항목을 선택합니다:
    • 생활용수: 20개 항목
    • 농업용수: 15개 항목
  4. 등록 버튼을 클릭합니다

🐄 5. 퇴·액비 시료 접수

가축분뇨 퇴비의 품질 검사를 위한 시료를 접수합니다.

퇴·액비 시료 접수 화면

접수 방법

  1. 생산자/업체명을 입력합니다
  2. 퇴비 종류(돈분, 우분, 계분, 혼합 등)를 선택합니다
  3. 생산지 주소를 입력합니다
  4. 검사 목적(품질검사, 부숙도검사, 성분분석 등)을 선택합니다
  5. 등록 버튼을 클릭합니다

⚠️ 6. 중금속 시료 접수

토양 중금속 오염 검사를 위한 시료를 접수합니다.

중금속 시료 접수 화면

접수 방법

  1. 시료 종류(토양, 농산물, 용수 등)를 선택합니다
  2. 채취 위치를 입력합니다
  3. 검사 항목(납, 카드뮴, 비소, 수은, 크롬 등)을 선택합니다
  4. 오염 의심 원인을 선택합니다
  5. 등록 버튼을 클릭합니다

🏷️ 7. 라벨 인쇄

검사 결과 발송을 위한 주소 라벨을 인쇄합니다.

라벨 인쇄 화면

라벨 인쇄 방법

  1. 각 시료 접수 페이지에서 라벨 인쇄 버튼을 클릭합니다
  2. 인쇄할 데이터를 확인합니다
  3. 시작 위치를 선택합니다 (이미 사용한 라벨지 재활용 시)
  4. 인쇄 버튼을 클릭합니다

💡 팁

A4 라벨지 2열 9행(18개) 템플릿을 사용합니다. 동일인 중복은 자동으로 제거됩니다.

🔄 8. 자동 업데이트

프로그램이 새 버전으로 업데이트되면 자동으로 알림을 받을 수 있습니다.

업데이트 방법

  1. 프로그램을 실행하면 자동으로 새 버전을 확인합니다
  2. 새 버전이 있으면 백그라운드에서 자동 다운로드됩니다
  3. 다운로드가 완료되면 업데이트 알림창이 표시됩니다
  4. "재시작" 버튼을 클릭하면 즉시 업데이트가 적용됩니다
  5. "나중에"를 선택하면 다음 실행 시 자동으로 적용됩니다

💡 팁

자동 업데이트는 설치 파일(.exe)로 설치한 경우에만 작동합니다. ZIP 파일로 압축을 풀어서 사용하는 경우에는 수동으로 새 버전을 다운로드해야 합니다.

🌐 인터넷 연결 필요

업데이트 확인 및 다운로드를 위해 인터넷 연결이 필요합니다.

💾 데이터 유지

업데이트 시 기존에 저장된 시료 데이터는 그대로 유지됩니다.

✨ 9. 유용한 팁

💾 자동 저장

데이터는 자동으로 저장됩니다. 상단의 자동저장 토글로 켜고 끌 수 있습니다.

📅 연도별 관리

상단의 연도 선택으로 연도별 데이터를 구분하여 관리할 수 있습니다.

📥 데이터 내보내기

목록 화면에서 엑셀 내보내기 버튼으로 데이터를 Excel 파일로 저장할 수 있습니다.

🔍 검색 기능

목록 화면의 검색 기능으로 특정 시료를 빠르게 찾을 수 있습니다.

✅ 완료 체크

검사가 완료된 시료는 체크박스를 클릭하여 완료 상태로 변경할 수 있습니다.

📍 주소 검색

주소 입력 시 "주소 검색" 버튼으로 도로명 주소를 쉽게 찾을 수 있습니다.

⌨️ 단축키

Ctrl+S: 저장 | Ctrl+F: 검색 | Ctrl+P: 인쇄
