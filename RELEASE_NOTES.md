## 버전 히스토리

### v1.7.7 (2026-01-15) - 최신 버전
**토양 중금속 페이지 법인 정보 필드 추가**

## 설치 방법

1. 위 다운로드 링크에서 `sample-log-setup.exe` 파일 다운로드
2. 다운로드한 파일 실행
3. 설치 완료 후 프로그램 자동 실행

## 자동 업데이트

앱 실행 시 새 버전이 있으면 자동으로 업데이트 알림이 표시됩니다.
